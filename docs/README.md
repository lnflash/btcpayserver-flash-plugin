The development of the Flash BTCPayServer plugin is divided into three main phases:

1. **Phase 1**: Core Lightning Integration with Flash
2. **Phase 2**: NFC Card Integration
3. **Phase 3**: User Interface and Merchant Experience

Each phase is detailed in the [Implementation Plan](implementation-plan.md) document.
